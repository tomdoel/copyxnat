static get_disallowed_project_ids(server, label)[source]¶

Return arrays of project names and secondary IDs that cannot be used for the destination project because they are already in use by other projects on this server. If the project already exists then the name and ID it is currently using are allowed (ie they will not be included in the disallowed lists).

Parameters

  • server – the XnatServer from which to get the disallowed IDs

  • label – the label of the project which
